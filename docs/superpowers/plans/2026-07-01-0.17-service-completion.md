# 0.17 Service Completion Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Complete the 0.16 service-domain API by migrating legacy monolith REST to `/v1/api/system` and `/v1/api/processes`, aligning palm provider and MCP REST proxy, and generating OpenAPI from per-service registries.

**Architecture:** Extend existing `palm/services/system` and `execution/processes` with thin REST handlers; delete transitional `/v1/jobs`, `/v1/instances`, `/v1/plans` routes; update edge consumers. No new service domains.

**Tech Stack:** Python 3.11+, Palm `ServerRuntime`, `SystemService`, `ProcessExecutionService`, `pytest`, `just docs-check` / `just guard-common`

**Design spec:** [docs/superpowers/specs/2026-07-01-0.17-service-completion-design.md](../specs/2026-07-01-0.17-service-completion-design.md)

**Prerequisite:** 0.16.5 shipped (`RELEASE-0.16.5.md`)
