# 0.15.3 Legacy Removal Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Delete transitional aliases and remaining shims in 0.15.3 with no deprecation window.

**Architecture:** Remove wizard-prefixed `interactive_runtime` aliases, `create_cli_app`, and `ChildWizardCompletionHook`. Trim `serializers.py` to snapshot-only. Delete `explorer/collection_input.py` shim. REST schema dedup is **0.15.2** ([dedupe plan](2026-06-30-0.15.2-dedupe.md)); ruff + 0.15.0 tag is **0.15.0 hygiene** ([release plan](2026-06-30-0.15.0-release-hygiene.md)).

**Parent track:** [docs/superpowers/specs/2026-06-30-0.15-cleanup-track-design.md](../specs/2026-06-30-0.15-cleanup-track-design.md)

**Tech Stack:** Python 3.12, `DictStateSchema`, `CqrsSchemaRegistry`, pytest, ruff, `just check`.

**Design spec:** [docs/superpowers/specs/2026-06-30-0.15.3-legacy-cleanup-design.md](../specs/2026-06-30-0.15.3-legacy-cleanup-design.md)
